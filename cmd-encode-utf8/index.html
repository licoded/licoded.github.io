修改cmd控制台默认编码为utf-8

2020-03-19

打开注册表 计算机\HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Command Processor

添加 autorun字符串，值为 chcp 65001 && cls

  1. 使用 win+r 快捷键打开运行窗口，输入 regedit 接着使用回车打开注册表编辑器

  2. 打开 “Command Processor” 目录，新建字符串值

  3. 编辑新建的字符串值

有时候如果想要临时改回 GBK 编码，可以用 chcp 936 命令临时改变当前窗口的编码
